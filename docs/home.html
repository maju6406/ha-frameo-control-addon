üì¶ Installation

Deploy the API server using Docker:

# Using Docker Compose
docker-compose up -d

# Using Docker directly
docker run -d \
  --name frameo-adb-api \
  --privileged \
  -v /dev/bus/usb:/dev/bus/usb \
  -v $(pwd)/data:/data \
  -p 5000:5000 \
  frameo-adb-api
